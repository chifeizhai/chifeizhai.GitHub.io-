终端的环境变量如何修改？

查看环境变量

输入 echo $PATH

修改环境变量

  1. 创建文件夹（名称自定，以下以aa代替）
  2. 在文件夹中创建名为xx（即命令代码，文件名无需后缀）的文件
  3. 编辑xx文件，输入所执行的命令
  4. 输入 vim ~/.bashrc 然后按shift+A进入插入模式
  5. 添加 export PATH="$PATH:文件夹aa的路径"
  6. 输入 source ~/.bashrc 使其生效

给环境变量添加别名

  1. 输入 vim ~/.bashrc 然后按shift+A进入插入模式
  2. 输入 alias xx="xxx" 使其生效。（从此输入xx相当于输入xxx）

输入命令代码时提示 permission denied:xxx (缺失权限)

输入ls -al查看权限（r为读，w为写，x为执行，-为无权限），查看缺失的权限。

例如：rwxr-xr-x

（前三位字母为自己的权限，中间三位为同电脑的不同用户，后三位为其他人）

  • 输入 chmod +w xxx 给xxx命令添加写的权限
  • 输入 chmod -x xxx 给xxx命令减去执行的权限
  • 输入 chmod 777 xxx 任何人对xxx命令拥有读、写、执行的权限
  • 输入 sudo chmod +w /etc/bashrc 用管理员身份给 /etc/bashrc文件加写权限
