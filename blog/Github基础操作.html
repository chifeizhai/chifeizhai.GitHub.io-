Github基础操作

配置Github公钥

  1. 终端输入 ssh-keygen -t rsa -b 4096 -C "自己的邮箱地址"
  2. 一直按回车，生成 ~/.ssh/id_rsa.pub 公钥文件
  3. 输入 cat ~/.ssh/id_rsa.pub 查看密钥
  4. 将密钥添加到Github网站上即可，步骤如下图

clone到本地

  1. 终端进入想存放文件的文件夹
  2. 输入 git clone 网站上的SSH链接 即可完成克隆

提交到本地仓库

  1. 输入 git add . (如出现warning,执行 git config --global core.autocrlf input 即可)
  2. 输入 git commit -am “ 描述信息 ”
  • 第一次执行commit可能不会成功出现如下报错

按提示依次执行 git config --global user.name “ 邮箱地址 ”


执行git config --global user.name “ 名字”

推送到远程仓库

执行 git push origin master

更新本地仓库

执行 git pull origin

本地创建项目上传至Github

  1. 在Github上创建一个新的空项目，获取他的ssh地址
  2. 在本地创建一个文件夹，名字随便取
  3. 进入文件夹，执行 git init
  4. 编辑文件夹内内容
  5. 执行 git add . 然后执行 git commit -am"描述"
  6. 执行git push ssh地址 master 完成上传

删除文件

  1. 执行 rm 文件名
  2. 执行 git commit -am"描述"

本地保留文件，删除远程仓库的文件

    1. 执行 git rm --cached 文件名
    2. 执行 git commit -m "删除无关文件"（无须执行 git add .）即可

    1. 执行 touch .gitignore 创建gitignore文件夹
    2. 执行 echo 文件名 > .gitignore 把文件加入该文件夹
    3. 执行 git add . 然后执行 git commit -am"描述"
    4. 执行 git push origin master 上传，从此gitignore内的文件不再上传
